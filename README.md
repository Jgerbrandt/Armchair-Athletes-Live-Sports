# ArmChairAthletes Live Sports <img style="float: right;" src="client/src/assets/media/logo.png" alt= “a” width="50" height="50">

This is a web application for viewing live sports results for different hockey leagues, including score, standings, and schedules. A user can create an acount and chose to follow their favourite team.
